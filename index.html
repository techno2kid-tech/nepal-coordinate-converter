Conversion Pipeline
DMS ↔ WGS84: Simple degree arithmetic. DMS → DD: DD = D + M/60 + S/3600
WGS84 ↔ MUTM: (1) Convert WGS84 DD to WGS84 ECEF XYZ. (2) Apply inverse Molodensky shift (ΔX,ΔY,ΔZ) to obtain Everest ECEF. (3) Convert Everest ECEF back to geodetic lat/lon using Everest ellipsoid. (4) Project onto MUTM using Transverse Mercator (CM=81/84/87, k₀=0.9999, FE=500000).
Reverse: MUTM Everest → geodetic Everest → ECEF Everest → apply +Molodensky → WGS84 ECEF → WGS84 geodetic.
